# useAlert

The useAlert hook is used to create a modal that can be used to display alerts.

```ts
function Component() {
    const { alert } = useAlert();

    return <Button onClick={() => alert('Hello world!')}>Alert</Button>
}
```

### useAlert returns an object with the following properties:
- **function alert(title: string, text: string, icon: AlertIconType, settings?: IAlertSettings) => Promise&lt;IAlertResult&gt;** - A function that can be used to add an alert to the modal.
- **function closeAlert** - A function that can be used to cancel an active alert.

### Alert Function Params

|Param|Type|Default Value|Description|Required|
|---|---|---|---|---|
|icon|string|success \| error \| warning \| info \| question|The icon to display in the alert.|false|
|title|string|undefined|The title of the alert.|false|
|text|string|undefined|The text of the alert.|false|
|settings|IAlertSettings(Object)||Settings for alert.|false|

#### IAlertSettings(Object)

|Property|Type|Default Value|Description|Required|
|---|---|---|---|---|
|showCancelButton|boolean|false|Show a cancel button.|false|
|showCloseButton|boolean|false|Show a close button.|false|
|showConfirmButton|boolean|true|Show a confirm button.|false|
|showDenyButton|boolean|false|Show a deny button.|false|
|cancelButtonText|string|'Cancel'|The text to display on the cancel button.|false|
|confirmButtonText|string|'Confirm'|The text to display on the confirm button.|false|
|denyButtonText|string|'Deny'|The text to display on the deny button.|false|
|focusCancel|boolean|false|Focus the cancel button.|false|
|focusConfirm|boolean|false|Focus the confirm button.|false|
|focusDeny|boolean|false|Focus the deny button.|false|
|reverseButtons|boolean|false|Reverse the buttons.|false|

### useAlert.setDefaultButtonsText

In order to allow an easier and more practical internationalization we have a static function in the hook "useAlert.setDefaultButtonsText".
You can call this function inside or outside the react context, thus facilitating internationalization.

```ts

useAlert.setDefaultButtonsText({
  cancelButtonText: 'Cancel Button',
  confirmButtonText: 'Confirm Button',
  denyButtonText: 'Deny Button',
});

```
