# useAlert

The useAlert hook is used to create a modal that can be used to display alerts.

## Hook response (Object)
**useAlert** returns an object with the following properties:
- **function addAlert(IAlertItemRequest)** - A function that can be used to add an alert to the modal.
- **cancelAlert** - A function that can be used to cancel an active alert.

### IAlertItemRequest Object

|Property|Type|Default Value|Description|Required|
|---|---|---|---|---|
|icon|string|'info'|The icon to display in the alert.|false|
|title|string|''|The title of the alert.|false|
|text|string|''|The text of the alert.|false|
|settings|settings(Object)||The settings of the alert.|false|

#### settings(Object)

|Property|Type|Default Value|Description|Required|
|---|---|---|---|---|
|showCancelButton|boolean|false|Show a cancel button.|false|
|showCloseButton|boolean|false|Show a close button.|false|
|showConfirmButton|boolean|true|Show a confirm button.|false|
|showDenyButton|boolean|false|Show a deny button.|false|
|cancelButtonText|string|'Cancel'|The text to display on the cancel button.|false|
|confirmButtonText|string|'Confirm'|The text to display on the confirm button.|false|
|denyButtonText|string|'Deny'|The text to display on the deny button.|false|
|focusCancel|boolean|false|Focus the cancel button.|false|
|focusConfirm|boolean|false|Focus the confirm button.|false|
|focusDeny|boolean|false|Focus the deny button.|false|
|reverseButtons|boolean|false|Reverse the buttons.|false|
