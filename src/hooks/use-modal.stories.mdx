# useModal

The useModal hook is used to handle the modal rendering using context provided by `<ApplicationProvider/>`, which is used to provide the modal state and actions.
The useModal hook can stack multiple modals on top of each other.


## Hook response (Object)
**useModal** returns an object with the following properties:
- **isOpen**: Boolean, whether the modal is open or not.
- **showModal**: `function(React.ComponentType<Props>, props: Props) => Promise`, show the provided modal component.
- **closeModal**: Function, closes active modal.
