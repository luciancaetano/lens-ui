# ApplicationProvider

The `ApplicationProvider` is a wrapper component that provides the `contexts` used by `useAlert`and `useToast` hooks.

### Properties

|Property|Type|Default Value|Description|Required|
|---|---|---|---|---|
|deviceContextDebounceTimmer|number|100|The debounce time in milliseconds for the device context.|No|
|toastsPlacement|`top-left`, `top-right`, `bottom-left`<br/>`bottom-right`, `top-center`, `bottom-center`|'bottom-right'|The placement of the toasts.|No|
