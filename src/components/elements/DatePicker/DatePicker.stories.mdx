# DatePicker

This is a small emblem commonly used to represent tags.

<Canvas>
    <DatePicker type="date" defaultValue={new Date()}/>
</Canvas>

### Base Properties

|Property|Type|Default Value|Description|Required|
|---|---|---|---|---|
|type|`date`, `month`, `range`, `year`, `multiple`, `week` | undefined | The type of the badge.|No|
|testingID| string | - | The id used for testing purposes.<br/>`<div data-testid="my-test-id"/>` |No|
|name| string | - | The name of the input.|No|
|placeholder| string | - | The placeholder of the input.|No|
|readOnly| boolean | false | If the input is read only.|No|
|ariaLabel| string | - | The aria label of the input.|No|
|disabled| boolean | false | If the input is disabled.|No|
|isError| boolean | false | If the input is in error state.|No|
|required| boolean | false | If the input is required.|No|
|isMobile| boolean | false | If the input is in mobile mode.|No|
|inline| boolean | false | If the input is in inline mode.|No|
|locale| IDatePickerLocale | - | The locale of the input.|No|
|onPickerOpen| `function()` | - | The function called when the calendar is opened.|No|
|onPickerClose| `function()` | - | The function called when the calendar is closed.|No|
|children| function (strDate: string, openCalendar: Function, handleValueChange: ChangeEvent): ReactNode | - | The function called when the calendar is closed.|No|
