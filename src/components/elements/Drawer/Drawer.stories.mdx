# Drawer

This is a small emblem commonly used to represent tags.

export const ToggleButton = ({children}) => {
  const [open, setOpen] = useState(false);
  const [side, setSide] = useState('left');
  return children({open, setOpen, side, setSide});
}

### Properties

|Property|Type|Default Value|Description|Required|
|---|---|---|---|---|
|children|React.ReactNode|-|Drawer content|false|
|isOpen|boolean|false|Open drawer|false|
|onClose|() => void|undefined|Close drawer|false|
|placement|'left' \| 'right' \| 'top' \| 'bottom'|'left'|Drawer placement|false|
|size|'small' \| 'medium' \| 'fullscreen' \| string \| number|'medium'|Drawer size|false|
|title|React.ReactNode|-|Drawer title|false|
|closeIcon|React.ReactNode|-|Close icon|false|
|testingID| string | - | The id used for testing purposes.<br/>`<div data-testid="my-test-id"/>` |No|
