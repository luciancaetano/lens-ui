# BottomNavigation

This is a small emblem commonly used to represent tags.

### Properties

#### BottomNavigation

|Property|Type|Default Value|Description|Required|
|---|---|---|---|---|
|activeId| string | - | id of active item |No|
|defaultActiveId| string | - | default id of active item |No|
|keepLabel| string | - | keep's action label when item not selected |No|
|onChange| Function(id: string) | - | change handler |No|
|testingID| string | - | The id used for testing purposes.<br/>`<div data-testid="my-test-id"/>` |No|

#### BottomNavigation.Action

|Property|Type|Default Value|Description|Required|
|---|---|---|---|---|
|id| string | - | id of active item, this must be unique per BottomNavigation |No|
|icon| React.ReactNode | - | Icon of Action |No|
|testingID| string | - | The id used for testing purposes.<br/>`<div data-testid="my-test-id"/>` |No|
