### List component Properties

|Property|Type|Default Value|Description|Required|
|---|---|---|---|---|
|intent|IntentType|primary|The intent of the list.|No|
|prefixMinWidth|number \| string|undefined|Min width of prefix container.|No|
|suffixMinWidth|number \| string|undefined|Min width of suffix container.|No|
|border|boolean|true|Render left and right border.|No|
|testingID| string | undefined | The id used for testing purposes.<br/>`<div data-testid="my-test-id"/>` |No|

### List.Item component Properties

|Property|Type|Default Value|Description|Required|
|---|---|---|---|---|
|intent|IntentType|'none'|The intent of the list item|No|
|isActive|boolean|false|Whether the list item is active|No|
|divider|boolean|false|Set item to divider (**no children rendering**)|No|
|heading|boolean|false|Whether the list item is a heading|No|
|prefix|React.ReactNode|undefined|Prefix elements, ex you can put an icon|No|
|suffix|React.ReactNode|undefined|Suffix elements, ex you can put an icon|No|
|prefixMargin|number \| string|undefined|Set's margin from content to prefix|No|
|suffixMargin|number \| string|undefined|Set's margin from content to suffix|No|
|suffix|React.ReactNode|undefined|Suffix elements, ex you can put an icon|No|
|cursor|string|default|Set cursor|No|
|testingID| string | undefined | The id used for testing purposes.<br/>`<div data-testid="my-test-id"/>` |No|
