## Properties

|Property|Type|Default Value|Description|Required|
|---|---|---|---|---|
|vertical|boolean|false|Whether the tabs should be vertical.|No|
|initialActiveTab|string|undefined|The initial active tab.|No|
|tabId|string|undefined|Set unique tab id.|Yes|
|activeTab|string|undefined|The active tab.|No|
|onTabClose|`function(tabId, MouseEvent)`|undefined|Handle tab close event.|No|
|tabBarExtraContentLeft|React.ReactNode|undefined|Left extract content|No|
|tabBarExtraContentRight|React.ReactNode|undefined|Right extract content|No|
|onChange|`function(tabId, MouseEvent)`|undefined|The callback to call when the active tab changes.|No|
|children|`function (ITabsItem \| null): React.ReactNode`|-|The callback to render the tab content.|Yes|
|testingID| string | - | The id used for testing purposes.<br/>`<div data-testid="my-test-id"/>` |No|

### Tabs.Tab

|Property|Type|Default Value|Description|Required|
|---|---|---|---|---|
|title|React.ReactNode|null|The title of the tab.|No|
|description|React.ReactNode|null|Description of the tab.|No|
|icon|React.ReactNode|null|Icon of the tab.|No|
|closeIcon|React.ReactNode|null|Define tab close icon.|No|
|closable|React.ReactNode|null|Define tab can show close icon and trigger close event.|No|
|tabId|string|-|The id of the tab.|Yes|
|tabRender|`function(props: IBaseTabProps) => React.ReactNode`|-|Custom tab pane render.|Yes|
|className|string|null|The class name of the tab.|No|
