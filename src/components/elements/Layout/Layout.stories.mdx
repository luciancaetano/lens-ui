The Layout component's handles application layout.

### Layout properties
The properties of Layout extends the `<div>` element.

|Property|Type|Default Value|Description|Required|
|---|---|---|---|---|
|testingID| string | undefined | The id used for testing purposes, `<div data-testid="my-test-id"/>` |No|

### Layout.Content properties
The properties of Layout.Content extends the `<div>` element.

|Property|Type|Default Value|Description|Required|
|---|---|---|---|---|
|layout|`horizontal`, `vertical`|`vertical`|The layout orientation.|Yes|
|testingID| string | undefined | The id used for testing purposes.<br/>`<div data-testid="my-test-id"/>` |No|

### Layout.SideBar properties
The properties of Layout.SideBar extends the `<aside>` element.

|Property|Type|Default Value|Description|Required|
|---|---|---|---|---|
|testingID| string | undefined | The id used for testing purposes, `<div data-testid="my-test-id"/>` |No|


### CSS Variables
The following CSS variables are available for use in the Layout component.

|Variable|Description|Default Value|
|---|---|---|
|**--lens-ui-app-layout-width**|The width of the app layout|`100vw`|
|**--lens-ui-app-layout-height**|The height of the app layout|`100vh`|
