### Properties

|Property|Type|Default Value|Description|Required|
|---|---|---|---|---|
|variant|string|'body1'|The variant of the typography.|Yes|
|align|string|'inherit'|The alignment of the typography.|No|
|noWrap|boolean|false|Whether or not the typography should wrap.|No|
|intent|string|null|The intent of the typography.|No|
|gutterBottom|boolean|false | Whether to add a bottom margin.|No|
|testingID| string | - | The id used for testing purposes.<br/>`<div data-testid="my-test-id"/>` |No|
