## Components properties

### Row Properties

|Property|Type|Default Value|Description|Required|
|---|---|---|---|---|
|start| Array&lt;xs \| sm \| md \| lg&gt; | - | Align elements to the start of the row |No|
|center| Array&lt;xs \| sm \| md \| lg&gt; | - | Align elements to the center of the row |No|
|evenly| Array&lt;xs \| sm \| md \| lg&gt; | - | Distribute items evenly |No|
|end| Array&lt;xs \| sm \| md \| lg&gt; | - | Align elements to the end of the row |No|
|top| Array&lt;xs \| sm \| md \| lg&gt; | - | Align elements to the top of the row |No|
|middle| Array&lt;xs \| sm \| md \| lg&gt; | - | Align elements to the middle of the row |No|
|bottom| Array&lt;xs \| sm \| md \| lg&gt; | - | Align elements to the bottom of the row |No|
|around| Array&lt;xs \| sm \| md \| lg&gt; | - | Distribute the contents around of a row or column |No|
|between| Array&lt;xs \| sm \| md \| lg&gt; | - | Distribute the contents between of a row or column |No|
|reverse| Array&lt;xs \| sm \| md \| lg&gt; | - | Reverse elements distribuition |No|
|testingID| string | - | The id used for testing purposes.<br/>`<div data-testid="my-test-id"/>` |No|

### Col Properties

|Property|Type|Default Value|Description|Required|
|---|---|---|---|---|
|first| Array&lt;xs \| sm \| md \| lg&gt; | - | order col at first |No|
|last| Array&lt;xs \| sm \| md \| lg&gt; | - | order col at last |No|
|xs| string(1...12) | - | xs breaking points |No|
|sm| string(1...12) | - | sm breaking points |No|
|md| string(1...12) | - | md breaking points |No|
|lg| string(1...12) | - | lg breaking points |No|
|offsetXs| string(1...12) | - | offset of col im xs screens |No|
|offsetSm| string(1...12) | - | offset of col im sm screens |No|
|offsetMd| string(1...12) | - | offset of col im Md screens |No|
|offsetLg| string(1...12) | - | offset of col im Lg screens |No|
|testingID| string | - | The id used for testing purposes.<br/>`<div data-testid="my-test-id"/>` |No|

### Container Properties

|Property|Type|Default Value|Description|Required|
|---|---|---|---|---|
|fluid| boolean | - | set container to fluid |No|
|testingID| string | - | The id used for testing purposes.<br/>`<div data-testid="my-test-id"/>` |No|
