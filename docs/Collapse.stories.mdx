### Collapse props

|Property|Type|Default Value|Description|Required|
|---|---|---|---|---|
|children| React.ReactNode | - | The children of the component. |No|
|activeId| string[] | - | The id of the panel that is active. |No|
|defaultActiveId| string[] | - | The id of the panel that is active by default. |No|
|onChange| (id: string[]) => void | - | The callback that is called when the active panel is changed. |No|
|expandIcon| (isActive: boolean) => React.ReactNode | - | The icon to use for the expand icon. |No|
|singleExpand| boolean | false | If true, only one panel can be expanded at a time. |No|
|testingID| string | - | The id used for testing purposes.<br/>`<div data-testid="my-test-id"/>` |No|

### Collapse.Panel props

|Property|Type|Default Value|Description|Required|
|---|---|---|---|---|
|header| React.ReactNode | - | The header of the panel. |Yes|
|id| string | - | The id of the panel. |Yes|
|children| React.ReactNode | - | The children of the panel. |No|
|testingID| string | - | The id used for testing purposes.<br/>`<div data-testid="my-test-id"/>` |No|
